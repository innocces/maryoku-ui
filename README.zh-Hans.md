### 👨‍🔧 快速开始

- **npm**

```bash
$ npm i @maryoku-ui/{{target}}
```

- **yarn**

```bash
$ yarn add @maryoku-ui/{{target}}
```

- **pnpm**

```bash
$ pnpm add @maryoku-ui/{{target}}
```

### 👻 参与贡献

项目使用**pnpm**进行管理. 并确保 **>= 7.x**

[贡献指南](https://github.com/maryoku-ui/maryoku-ui/blob/main/CONTRIBUTING.md)

组件库编译

```bash
$ pnpm run build
```

文档预览

```bash
$ pnpm run doc:dev
```
